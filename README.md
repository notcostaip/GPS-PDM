Um aplicativo de GPS moderno e imersivo construído com **Next.js** e **Leaflet**. Ele rastreia sua localização em tempo real, mostra a precisão do sinal e permite descobrir estabelecimentos ao seu redor em um mapa Dark Mode elegante com navegação de um dedo. 📍

</div>

<br/>

---

<div align="center">

### &nbsp; 🎬 Recursos

</div>

O aplicativo utiliza a API de Geolocalização do navegador para rastrear precisamente suas coordenadas geográficas e exibi-las de forma visualmente imersiva.

| Funcionalidade | Descrição |
|-------|--------------|
| 🎯 **Precisão em Tempo Real** | Mostra latitude, longitude e margem de erro. |
| 🗺️ **Mapa Dark Imersivo** | Interface com CartoDB Dark Matter para navegação noturna. |
| 📍 **Pontos de Interesse** | Simula a descoberta de postos, restaurantes, farmácias, etc. |
| 📱 **Responsividade** | 100% otimizado para celulares com navegação fluida de 1 dedo. |

<div align="center">

### &nbsp; 🚀 Como Rodar

</div>

1. Clone o repositório e instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra [http://localhost:3000](http://localhost:3000) com o seu navegador e permita o acesso à sua localização.
